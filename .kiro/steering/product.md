# Product Overview

ScoreBar is a baseball game management and live streaming overlay application. It provides real-time scorekeeping, player statistics tracking, and broadcast-ready overlays for baseball games.

## Core Features

- Real-time game state management (innings, balls, strikes, outs, base runners)
- Team and player lineup management with batting order
- Live game statistics tracking (hits, runs, errors, stolen bases, etc.)
- Broadcast overlays (scoreboards, scorebugs, formations, player stats)
- Banner and sponsor management for broadcasts
- Multi-game support with game history
- Real-time synchronization via WebSocket connections
- User authentication and organization management

## Key Domain Concepts

- Baseball rules implementation (MLB Official Rules)
- Designated Hitter (DH) support
- Advanced baseball scenarios: dropped third strike, wild pitch, passed ball, stolen bases, balks
- Runner advancement validation (Rule 5.08(a) - third out scenarios)
- Property-based testing for correctness validation
